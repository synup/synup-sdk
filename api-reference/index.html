    synup.client.SynupClient ¶

    Client for the Synup v4 API. Create with an API key; all methods return structured data.

    Example

    client = SynupClient(api_key="YOUR_API_KEY") locations = client.fetch_all_locations(first=10) if locations.get("success"): for loc in locations["locations"]: print(loc["name"])

    synup.exceptions.SynupAPIError ¶

    Bases: Exception

    Raised when the API returns a non-2xx response. Use status_code and response_body to handle or display.

    Attributes:

    Name Type Description
    status_code

    HTTP status (e.g. 401, 404, 500). Use to branch: 401 → check API key; 404 → resource missing; 5xx → retry or contact support.
    response_body

    Raw response text from the API; often JSON with an error message or validation details.
    Example

    try: client.fetch_all_locations() except SynupAPIError as e: if e.status_code == 401: print("Invalid or missing API key. Check Settings → Integrations.") else: print(f"Request failed: {e.status_code} — {e.response_body}")
