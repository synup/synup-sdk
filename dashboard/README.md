# Synup SDK Explorer Dashboard

Interactive web UI to browse, test, and explore all Synup SDK endpoints.

## Setup

```bash
pip install -r requirements.txt
```

## Run

```bash
SYNUP_API_KEY=your_api_key streamlit run app.py
```

Opens at `http://localhost:8501`.

## Features

- Browse all SDK methods by category
- Search and filter endpoints
- Auto-generated input forms from method signatures
- Execute endpoints and see JSON responses
- Full documentation for each method
