SDK for Synup API

Build on Synup's
local presence infrastructure

Listings, reviews, and local search. One install, full API coverage, zero boilerplate.

Get started Try it live
quickstart.py 
from synup import SynupClient

client = SynupClient(api_key="YOUR_API_KEY")

# Fetch all locations
locations = client.fetch_all_locations(fetch_all=True)

# Get reviews for a location
reviews = client.fetch_interactions(16808, first=20)

# Respond to a review
client.respond_to_review(
    interaction_id="review-uuid",
    response_content="Thank you for your feedback!"
)

Built for developers

Full API coverage

Locations, listings, reviews, rankings, analytics, users, and more.

🔌

Thin wrapper

Returns raw API data. No opinionated models. Bring your own data layer.

📄

Auto-pagination

Pass fetch_all=True to any list method. The SDK handles cursor logic.

🔑

Auto-encoding

Pass numeric IDs like 16808. The SDK base64-encodes them automatically.

📖

Full docstrings

Every method has Args, Returns, and Example. IDE autocompletion just works.

🛠

Error handling

Clear exceptions with HTTP status and response body. Easy to debug.
